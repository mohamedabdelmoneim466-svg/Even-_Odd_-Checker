# Even / Odd Checker 🔢

A simple and interactive C++ console application that determines whether a given number is **Even** or **Odd**. It also includes input validation to handle non-numeric entries gracefully.

---

## 🚀 Features
* **User-Friendly Interface:** Clear instructions and decorative ASCII borders.
* **Input Validation:** Detects if the user enters something other than a number and displays an "Invalid Input" message.
* **Accurate Logic:** Uses the modulo operator (`%`) to determine parity.

## 🛠️ Built With
* **Language:** C++
* **IDE:** Visual Studio 2026

## 📖 How to Use
1.  Run the application.
2.  Enter any integer when prompted.
3.  The program will instantly tell you if the number is **Even** or **Odd**.
4.  If you enter a character or symbol, the program will catch the error.
